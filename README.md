Este proyecto consiste en la validación de una API REST mediante pruebas automatizadas para garantizar que los endpoints respondan correctamente bajo diferentes escenarios.

El objetivo fue asegurar que la API cumpliera con los requisitos funcionales y de negocio.

Identificación de endpoints críticos

Validación de códigos de respuesta HTTP

Verificación de estructura del JSON

Pruebas positivas y negativas

Manejo de datos inválidos
